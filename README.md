# FORMS
#### INSTALLATION :fire:

`composer require wahyu/forms`

#### REQUIREMENTS :warning: 

:black_small_square: Bootstrap 4.

#### INSTALL BY CLI :globe_with_meridians:

* [yarn](https://yarnpkg.com/) (recommended):

```
$ yarn add bootstrap
```

* [npm](https://www.npmjs.com/):

```
$ npm install bootstrap
```
